# Contributing to theblogship

Thanks for your interest in contributing! Contributions that improve content, accessibility, structure, or security are welcome.

What you can do
- Open issues for bugs or feature requests.
- Submit pull requests for fixes, improvements, or new content.

Getting started
1. Fork the repository and create a branch:

```bash
git checkout -b feature/brief-description
```

2. Make changes in a focused commit.
3. Push your branch and open a Pull Request against `main`.

Guidelines
- Keep changes small and focused.
- Do not commit secrets, credentials, or database dumps. Use `config.local.php` for local secrets.
- Follow existing code and markup style.
- Write descriptive commit messages and PR descriptions.

Reporting security issues
- Do not disclose security issues in a public issue. Email security-sensitive information to `iprouteumair@gmail.com`.

License
- By contributing, you agree that your contributions will be licensed under the project's MIT License.

Thank you — contributions keep this project healthy and useful.
